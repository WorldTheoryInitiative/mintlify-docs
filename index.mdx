## What is PolyRouter?

<Warning>
  **Open Beta:** PolyRouter is currently in open beta. All features are free with 10 req/sec rate limits. We're actively improving the platform based on your feedback!

PolyRouter provides **unified API access to 7 prediction market platforms**: Polymarket, Kalshi, Manifold Markets, Limitless, ProphetX, Novig, and SX.bet. Get real-time market data, sports betting odds, and historical prices through one simple API.

## Examples

### Fetch Markets

<CodeGroup>

```bash cURL
curl "https://lsplqyqiubvctfpfnukr.supabase.co/functions/v1/markets-v2?platform=polymarket&limit=5" \
  -H "X-API-Key: test_1234567890abcdef"
```

```javascript JavaScript
const response = await fetch(
  'https://lsplqyqiubvctfpfnukr.supabase.co/functions/v1/markets-v2?platform=polymarket&limit=5',
  { headers: { 'X-API-Key': 'test_1234567890abcdef' } }
);

const data = await response.json();
data.markets.forEach(market => {
  console.log(`${market.title}: ${(market.current_prices.yes.price * 100).toFixed(1)}%`);
});
```

</CodeGroup>

### Get NFL Odds

```bash
# Find today's games
curl "https://lsplqyqiubvctfpfnukr.supabase.co/functions/v1/list-games-v1?league=nfl&limit=5" \
  -H "X-API-Key: test_1234567890abcdef"

# Get odds for a specific game
curl "https://lsplqyqiubvctfpfnukr.supabase.co/functions/v1/games-v1/BUFvKC20251020" \
  -H "X-API-Key: test_1234567890abcdef"
```

<Info>
  **Game IDs** follow the format `{AwayTeam}v{HomeTeam}{YYYYMMDD}`. Use `/list-games-v1` to discover available games.
</Info>

### Query Price History

```bash
curl "https://lsplqyqiubvctfpfnukr.supabase.co/functions/v1/price-history-v2?market_ids=516710&interval=1h&limit=24" \
  -H "X-API-Key: test_1234567890abcdef"
```
