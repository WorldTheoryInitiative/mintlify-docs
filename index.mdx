## What is PolyRouter?

<Warning>
  **Open Beta:** PolyRouter is currently in open beta. All features are free with 10 req per minute rate limits. We're actively improving the platform based on your feedback! 

  IF YOU NEED HIGHER RATE LIMITS, OR HAVE ANY QUESTIONS, MESSAGE **@fieldviolence** ON X!

PolyRouter provides **unified API access to 7 prediction market platforms**: Polymarket, Kalshi, Manifold Markets, Limitless, ProphetX, Novig, and SX.bet. Get real-time market data, sports betting odds, and historical prices through one simple API.

## Examples

### Fetch Markets

<CodeGroup>

```bash cURL
curl "https://api.polyrouter.io/functions/v1/markets?platform=polymarket&limit=5" \
  -H "X-API-Key: 5fa709a5-0634-44c3-a991-57166d3c376d"
```

```javascript JavaScript
const response = await fetch(
  'https://api.polyrouter.io/functions/v1/markets?platform=polymarket&limit=5',
  { headers: { 'X-API-Key': '5fa709a5-0634-44c3-a991-57166d3c376d' } }
);

const data = await response.json();
data.markets.forEach(market => {
  console.log(`${market.title}: ${(market.current_prices.yes.price * 100).toFixed(1)}%`);
});
```

</CodeGroup>

### Get NFL Odds

```bash
# Find today's games
curl "https://api.polyrouter.io/functions/v1/list-games?league=nfl&limit=5" \
  -H "X-API-Key: 5fa709a5-0634-44c3-a991-57166d3c376d"

# Get odds for a specific game
curl "https://api.polyrouter.io/functions/v1/games/BUFvKC20251020" \
  -H "X-API-Key: 5fa709a5-0634-44c3-a991-57166d3c376d"
```

<Info>
  **Game IDs** follow the format `{AwayTeam}v{HomeTeam}{YYYYMMDD}`. Use `/list-games` to discover available games.
</Info>

### Query Price History

```bash
curl "https://api.polyrouter.io/functions/v1/price-history?market_ids=516710&interval=1h&limit=24" \
  -H "X-API-Key: 5fa709a5-0634-44c3-a991-57166d3c376d"
```
