## Authentication

All endpoints require an API key in the `X-API-Key` header:

```bash
curl "https://lsplqyqiubvctfpfnukr.supabase.co/functions/v1/markets-v2" \
  -H "X-API-Key: test_1234567890abcdef"
```

<Note>
  [Get your free API key](https://polyrouter.io) - takes 30 seconds.
</Note>

**Base URL:** `https://lsplqyqiubvctfpfnukr.supabase.co/functions/v1`
